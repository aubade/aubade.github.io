01 Solidworks » 01 Automated Macro Drawing

This is a test-case for a sheet metal properties macro written over the course of a week. Using info generated by solidworks, all of the information in the title block save for the Process and Title are automatically calculated. These values are re-computed whenever the part is rebuilt.

The die size in the bend notes is determined by reading values from an excel spreadsheet and is selected based on the material gauge.
