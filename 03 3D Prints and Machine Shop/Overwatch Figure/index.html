03 3D Prints and Machine Shop » Overwatch Figure

This was an attempt to make a paintable miniature based on an existing design. The first attempt to print at 1:1 model scale on a Flashforge 3D Printer yielded unsatisfactory results, so a second attempt was made to print it in segmented form on a Mojo printer, which was deemed a success.
